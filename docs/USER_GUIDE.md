## 快速开始

### 第一步：导入视频

1. 点击菜单栏 **文件 -> 导入视频**
2. 在弹出的对话框中点击"浏览"按钮
3. 选择要处理的视频文件（支持MP4、AVI、MOV等格式）
4. 输入项目名称（必填）
5. 输入项目描述（可选）
6. 点击"确定"按钮

### 第二步：分析视频

导入视频后，系统会自动开始分析。如果需要手动分析：

1. 点击菜单栏 **工具 -> 分析视频**
2. 等待进度条完成
3. 分析完成后，右侧会显示检测到的镜头列表

### 第三步：生成文案

1. 点击菜单栏 **工具 -> 生成文案**
2. 在弹出的对话框中选择：
   - 解说风格（剧情、搞笑、悬疑等）
   - 文案长度
   - 其他参数
3. 点击"生成"按钮
4. 等待AI生成完成
5. 在下方的文案编辑器中查看和编辑生成的文案

### 第四步：合成配音

1. 在右侧的"配音设置"面板中：
   - 选择音色
   - 调整语速、音调、音量
   - 点击"试听"按钮预览效果
2. 点击菜单栏 **工具 -> 合成配音**
3. 等待语音合成完成

### 第五步：导出视频

1. 点击菜单栏 **文件 -> 导出视频**
2. 在弹出的对话框中设置：
   - 输出路径
   - 视频格式
   - 视频质量
   - 分辨率和帧率
   - 是否保留原音
3. 点击"导出"按钮
4. 等待视频合成完成

## 功能详解

### 镜头检测

**自动检测模式**：
- 系统会自动识别视频中的镜头切换点
- 基于画面内容变化检测
- 可在设置中调整检测灵敏度

**手动调整**：
- 在镜头列表中选择镜头
- 在右侧"镜头编辑器"中可以：
  - 分割镜头
  - 合并镜头
  - 删除镜头

### 文案生成

**AI生成**：
- 支持多种解说风格
- 可设置文案长度
- 可添加额外的上下文信息

**手动编辑**：
- 在文案编辑器中直接修改
- 支持富文本格式
- 实时显示字数统计

**文案优化**：
- 点击"优化文案"按钮
- AI会自动优化语言表达
- 保持原意的同时提升质量

### 配音合成

**音色选择**：
- 支持多种中文音色
- 男声、女声可选
- 不同风格和特点

**参数调节**：
- **语速**：0.5x - 2.0x
- **音调**：低 - 高
- **音量**：0% - 200%

**预览试听**：
- 点击"试听"按钮
- 使用测试文本预览效果
- 调整满意后再批量合成

### 视频导出

**格式选择**：
- MP4（推荐）
- AVI
- MOV
- MKV

**质量设置**：
- **高质量**：适合最终发布
- **中等质量**：平衡质量和文件大小
- **低质量**：快速预览

**音频设置**：
- 可选择是否保留原视频音频
- 可调整原音音量（与解说混合）

## 常见问题

### Q: 视频分析失败怎么办？

A: 可能的原因和解决方法：
1. 视频格式不支持 → 转换为MP4格式
2. 视频文件损坏 → 使用其他视频
3. 内存不足 → 关闭其他程序
4. FFmpeg未安装 → 安装FFmpeg

### Q: AI生成的文案不满意？

A: 可以尝试：
1. 调整解说风格
2. 添加更多上下文信息
3. 手动编辑优化
4. 重新生成

### Q: 配音效果不好？

A: 建议：
1. 尝试不同的音色
2. 调整语速和音调
3. 优化文案的断句
4. 使用标点符号控制停顿

### Q: 导出的视频很大？

A: 可以：
1. 降低视频质量设置
2. 降低分辨率
3. 降低帧率
4. 使用更高效的编码格式

## 技巧和建议

### 提高文案质量

1. **提供清晰的上下文**：告诉AI视频的主题和目标受众
2. **选择合适的风格**：根据视频类型选择解说风格
3. **适当的长度**：不要太长或太短
4. **多次生成对比**：生成多个版本选择最好的

### 优化配音效果

1. **选择合适的音色**：与视频内容匹配
2. **调整语速**：根据视频节奏调整
3. **注意停顿**：使用标点符号控制
4. **背景音乐**：适当添加背景音乐增强效果

### 提高工作效率

1. **使用快捷键**：熟悉常用快捷键
2. **保存项目**：及时保存避免丢失
3. **批量处理**：一次处理多个镜头
4. **预设模板**：保存常用设置
